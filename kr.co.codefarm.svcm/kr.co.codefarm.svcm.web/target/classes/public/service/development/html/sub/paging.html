커스텀 엘리먼트

  • paging 커스텀 엘리먼트는 필수적으로 tagset 항목과 rowposition 항목을 제공해야 합니다.
  • emptynotice 항목은 데이터가 존재하지 않을 시 "검색결과가 없습니다." 라는 메시지의 표시 여부를 나타냅니다. (false로 지정해야 나타나지 않음)
  • pagingset: new PagingSet() , type : String ('normal')

테스트 데이터

  • 데이터는 전체 건수를 지정합니다. this.PagingSet.setTotalCnt([총 건수:Integer]);
  • vue의 watch 기능으로 PagingSet.currentPage를 감시하여 페이지 변경 시 데이터를 조회 할 수 있습니다.
  • PagingSet 의 상세 내용은 Dataset > PagingSet 을 참고 바랍니다.
  • PagingSet 생성 시 파라메터는 (pPageCnt:Integer, pRowCnt:Integer) 로 화면에 나타날 페이지 번호 개수, 및 한 페이지당 로우 건수 입니다.
