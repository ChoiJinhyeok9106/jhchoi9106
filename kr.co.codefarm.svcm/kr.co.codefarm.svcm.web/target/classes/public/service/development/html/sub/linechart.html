Toast UI Chart-Line

[Line chart] basic

  • http://nhnent.github.io/tui.chart/latest/tutorial-example03-01-line-chart-basic
  • 사용되는 화면: 실적관리>전체실적확인>역량성장도[탭]
  • 1. 차트를 그릴 위치에 div를 생성하고 id 값을 설정
  • 2. 설정한 id 값을 var container = document.getElementById('설정한 id'); 에 입력 - 아래의 예제 소스 1번줄
  • 3. var container: 차트를 그릴 위치 지정
  • 4. var data: 차트의 데이터
  • 5. var options: 차트의 옵션 설정 - https://github.com/nhnent/tui.chart/blob/v3.6.0/src/js/chart.js - line chart로 검색
  • 6. tui.chart.lineChart(container, data, options); - 차트 그리기
