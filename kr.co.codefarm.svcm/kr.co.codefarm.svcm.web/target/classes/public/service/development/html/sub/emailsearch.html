이메일 작성 팝업

  • 이메일 작성 팝업 활용법
  • ref를 통한 오픈함수를 이용해 팝업을 오픈합니다.
    ex) this.$refs.popupemail.showEmailPopup();
  • 'id'를 지정하여 사용합니다.
  • :email 에 이메일 주소를 콤마(,)로 구분하여 문자열로 지정할 경우 팝업에 받는 사람 목록에 자동으로 추가 됩니다.
  • :modifiable 을 false로 지정할 파라메터로 지정 된 email목록을 수정할 수 없게됩니다.
  • :fixedsender 을 true로 지정할 경우 보내는 사람을 수정할 수 없습니다.
  • :sender 에 보내는 사람 메일 주소를 입력하여 미리 셋팅할 수 있습니다.
  • :placeholder 항목에 이메일 작성 에디터의 placeholder 내용을 지정할 수 있습니다.
