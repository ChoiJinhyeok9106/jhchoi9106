[Column Chart] basic

  • http://nhn.github.io/tui.chart/latest/tutorial-example02-01-column-chart-basic
  • 사용되는 화면: 성과관리>지표코드관리>성과달성현황
  • 1. 차트를 그릴 위치에 div를 생성하고 id 값을 설정
  • 2. 설정한 id 값을 var container = document.getElementById('설정한 id'); 에 입력 - 아래의 예제 소스 1번줄
  • 3. var container: 차트를 그릴 위치 지정
  • 4. var data: 차트의 데이터
  • 5. var options: 차트의 옵션 설정 - https://github.com/nhnent/tui.chart/blob/v3.6.0/src/js/chart.js - column chart로 검색
  • 6. tui.chart.columnChart(container, data, options); - 차트 그리기
