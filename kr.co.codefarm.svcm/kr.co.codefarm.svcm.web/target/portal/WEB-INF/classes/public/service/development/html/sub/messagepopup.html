메세지 팝업

  • 메세지 팝업 활용법
  • modal-message 태그로 사용
  • 태그 호출 시 :modalset 지정(vue의 data영역 messageModal: new ModalSet('[id:String]') 형태로 사용)
  • ModalSet의 openModal('[Type:String]', [Title:String], [Contents:String], [Size:String], [CURRENT_ID:String], [CURRENT_DATA]) 메소드로 메시지 오픈 가능
  • [Type:String] : Message Modal 의 종류를 지정, 종류로는 message, confirm 이 있습니다. (나타나는 버튼 차이)
  • [Title:String] : 팝업의 제목
  • [Contents:String] : 팝업의 내용
  • [Size:String] : 팝업의 크기(small, middle, large)
  • [CURRENT_ID:String] : 현재 오픈되는 팝업의 ID (닫기나 확인 버튼 눌릴 시 아이디가 반환)
  • [CURRENT_DATA] : 현재 오픈되는 팝업의 추가정보 (닫기나 확인 버튼 눌릴 시 해당 정보가 반환)
  • messagepoppupcallback 함수는 콜백 함수로, 콜백이 필요할 경우 v-on:messagepoppupcallback=["부모창의 콜백함수"] 형태로 지정해야 합니다.
  • 콜백함수는 3가지 파라메터로 구성되어있습니다.
  • [1] btnTyep : 어떤 버튼을 클릭하여 팝업을 종료하였는지를 나타냅니다.('confirm','cancel','close')
  • [2] id : 팝업 오픈 시 지정 된 id를 반환합니다.
  • [3] data : 팝업 오픈 시 지정 된 data를 반환합니다.
