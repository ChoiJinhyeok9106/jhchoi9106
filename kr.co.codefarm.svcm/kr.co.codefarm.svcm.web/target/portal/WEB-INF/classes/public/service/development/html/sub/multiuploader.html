커스텀 엘리먼트

  • file-uploader 커스텀 엘리먼트는 한 번에 드래그앤 드롭 혹은 파일찾기 창에서 다건의 파일을 선택하여 업로드 가능하며 시각적인 요소가 포함 된 템플릿 입니다.
  • fileset : new FileSet (필수) 입니다.
  • type : String 데이터로 'list'와 'img'로 지정 가능합니다.
  • fileid : 파일과 관련된 테이블 명입니다.
  • filekey : 파일과 관련 된 테이블의 PK 데이터 정보(컬럼이 2개 이상일 경우 #을 구분자로 공백없이 제공)
  • filehangmok : 등록하려는 파일의 종류 입니다. (String) -> 작성자가 임의로 지정 가능합니다.
  • title : 파일 템플릿의 Title 명 입니다.
  • modify : 등록 및 삭제가 가능한 상태인지의 여부 입니다.(Boolean)
  • floatnone : float : none
  • imgwidth : 이미지의 가로 사이즈

테스트 데이터

  • 해당 파일 엘리먼트에 파일리스트를 가져오려할 경우 아래와 같이 사용해야 합니다.
  • this.FileSet.getFiles([테이블명:String], [해당테이블의 키값:String(구분자 #)], [파일의 종류:String], [파일시퀀스:Integer -> null일경우 다건 조회 됨]);
  • FileSet 선언 시 : new FileSet('ADD', 10) 형태로 서비스 명과 제한 카운터를 입력합니다.
  • FileSet 의 상세 내용은 Dataset > FileSet 을 참고 바랍니다.
