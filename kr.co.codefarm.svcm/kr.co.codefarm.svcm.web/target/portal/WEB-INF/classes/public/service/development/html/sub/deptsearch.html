팝업-부서 검색 팝업

부서 검색 팝업

  • 부서 검색 팝업 활용법
  • popup-dept 태그로 사용, callback 함수에서 리턴되는 데이터를 확인, 팝업 오픈이벤트를 활용하기 위한 ref지정
  • lovcomp 값을 true로 입력 할 경우 아래 예제와 같이 돋보기를 포함한 검색 컴포넌트가 제공됩니다.
  • lovcomp 값을 false로 입력 할 경우 별도의 컴포넌트가 제공 되지 않고 ref를 통한 오픈함수를 이용해 팝업을 오픈합니다.
    ex) this.$refs.popupdept.showDeptPopup();
  • type값을 입력하면 부서검색 팝업의 종류를 구분하여 오픈 합니다. 1~4 가 존재합니다.(본부행정(1), 대학원(2), 대학(학부)(3), 기타부속기관(4)) -> 문자열로 입력, '' 혹은 null로 전송할 경우 전체 검색
  • id : 고유 아이디
