[Radial chart] basic

  • 1. 차트를 그릴 위치에 div를 생성하고 id 값을 설정

  • 3. var container: 차트를 그릴 위치 지정

  • 5. var options: 차트의 옵션 설정 - https://github.com/nhnent/tui.chart/blob/v3.6.0/src/js/chart.js - radial chart로 검색
  • 6. tui.chart.radialChart(container, data, options); - 차트 그리기
