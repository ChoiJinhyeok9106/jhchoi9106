  • 전국 초·중·고 검색 팝업 활용법
  • ref를 통한 오픈함수를 이용해 팝업을 오픈합니다.
    ex) this.$refs.popupallschool.showAllSchoolPopup();
  • 'id'를 지정하여 사용합니다.
  • :gubun 은 초등학교, 중학교, 고등학교를 구분합니다.
    ex) elem_list : 초등학교, midd_list : 중학교, high_list : 고등학교
  • :region 은 지역을 나타냅니다.
    ex) 100260 : 서울특별시, 100267 : 부산광역시, 100269 : 인천광역시, 100271 : 대전광역시, 100272 : 대구광역시, 100273 : 울산광역시, 100275 : 광주광역시, 100276 : 경기도, 100278 : 강원도, 100280 : 충청북도, 100281 : 충청남도, 100282 : 전라북도, 100283 : 전라남도, 100285 : 경상북도, 100291 : 경상남도, 100292 : 제주도
  • :searchargv 을 검색어를 나타냅니다.
  • :sender 에 보내는 사람 메일 주소를 입력하여 미리 셋팅할 수 있습니다.
